Werken met GIT

  • git init : git initialiseren
  • git status : status van het project bekijken
  • git add `files` : files toevoegen aan git
  • git add --all
  • git commit : versie bewaren ( i -> insert / tekstje schrijven / esc -> escape / :wq -> saven )
  • git commit -m "versie......"
